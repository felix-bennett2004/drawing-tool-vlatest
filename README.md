## Overview

Drawing Tool is a simple HTML-based web app designed for basic drawing tasks. It gives you a straightforward canvas for sketching without needing to install a desktop application, which makes it handy for fast doodles, practice sessions, and lightweight visual notes.

Since it operates in the browser, it is easy to open and share, and it works well on typical platforms with modern browser support. The project stays intentionally uncomplicated, prioritizing direct drawing over a full-featured editing environment.

4. If you are hosting it yourself, point your browser to the local or published site URL.
